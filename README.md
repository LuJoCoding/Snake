# Snake Game üêç

A classic Snake game implemented in Python using **Pygame**. Collect apples, avoid collisions, and try to beat your high score!

---

## Features

- Classic Snake gameplay
- Multiple apple types: **regular**, **golden apple** (speed boost and double score), and **slow apple** (x5 score and slow down)

---

## How to Play

Use the **W, A, S, D** keys to move the snake:

- **W** ‚Üí Up
- **A** ‚Üí Left
- **S** ‚Üí Down
- **D** ‚Üí Right

---

## Installation

### Windows

1. Download the ZIP file from this repository.
2. Extract the ZIP.
3. Run the `snake.exe` file in the folder.

### Linux / macOS

1. Download the ZIP file from this repository.
2. Extract the ZIP.
3. Make sure you have Python 3 installed.
4. Install the required packages using pip:

pip install pygame
